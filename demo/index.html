jQueryUI contextmenu demo

short description

Provides a simple wrapper for jQueryUI's menu (Please check documentation if you want to learn more on menus events). If you take care yourself bout the menu bindings this simple widget doesn't touch your menu layer.

options

  • pageX / pageY [ default 0 ]

    use that if you call open yourself

  • forceUrlHandling [ default true ]

    As the menu currently prevents anchor clicks you should keep that activate that option if you want urls to be loaded. In that case an url has to start with http. this option will be removed

  • menu [ jQuery object of domnode ]
  • menuOptions [default null; uses defaults if you don't provide anything]
  • position [ position rendering of contextmenu]
